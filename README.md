# Arenula

MCP (Model Context Protocol) server suite for the s&box game engine.

Based on [Ozmium MCP Server](https://github.com/ozmium7/ozmium-mcp-server-for-sbox) by ozmium7.

## Servers

| Server | Purpose | Runtime |
|---|---|---|
| **Editor** | Scene manipulation, editor control, compilation | C# (s&box plugin) |
| **API** | Offline type/member reference | Node.js |
| **Docs** | Narrative documentation | Node.js |
